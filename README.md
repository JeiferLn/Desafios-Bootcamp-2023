# Desafio 09/05

En esta desafio empezamos a hacer paginas webs un poco mas avanzadas donde aplicamos multiples etiquetas y estilos para llegar al resultado esperado.

## Aprendimos
* Hacer uso de las etiquetas ***form***, ***label***, ***input*** y ***button***.
* Aplicar estilos al scroll de la pagina web.
* Usar el  `Display:Flex` para colocar contenedores uno en frente de otro.
