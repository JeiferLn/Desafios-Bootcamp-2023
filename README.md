# Desafio 13/06

En este desafio, aplicamos por primera vez a nuestros proyectos el lenguaje de JavaScript para hacer nuestras paginas dinamicas

## Aprendimos
* Hacer lo basico de JS (declarar e inicializar variables, crear funciones, hacer operaciones matematicas y mostrar el resultado en la consola)
* Conectar nuestros *HTML* junto con nuestro *JS* para hacer las paginas mucho mas interactivas
* Usar todo lo aprendido hasta el momento para tener una pagina mucho mas limpia (HTML, CSS, JS, Bootstrap)
